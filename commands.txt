Pour installer hashlips engine (https://github.com/HashLips) :

	npm install


Pour générer les n NFTs :

	node index.js


Pour déployer (dans NFTS\appnfts) :

	npx hardhat run scripts/deploy.js --network rinkeby


Pour démarrer le serveur en local :

	npm start


Pour compiler le smartcontract solidiy (dans le dossier du contrat) :

	npx hardhat compile
